Pull Current Season from TheSportsDB

Free, no sign-up needed. TheSportsDB provides Ligat Ha'Al results for seasons the API-Football free plan doesn't cover. Click the button to pull and merge the season automatically.

Run this command in your terminal from the project folder: 

python pull_thesportsdb.py 2025

Change 2025 to the season start year if needed. The script pulls round-by-round, merges into the SQLite database, and rebuilds the calibration table automatically. Then run python calibrate.py to update the model weights.

Import FootyStats CSV (manual upload)

Alternative: If you have a FootyStats CSV, you can upload it here instead. Note that the CSV download on FootyStats requires a paid account.
Step 1 — Download the CSV from FootyStats
  1. Go to footystats.org/israel/premier-league
  2. Make sure you are on the correct season (check the season selector near the top of the page)
  3. Scroll down to the matches table and look for a Download CSV or Export button
  4. Save the file anywhere on your computer
Step 2 — Upload and import
e.g. 2025 for the 2025/26 season
